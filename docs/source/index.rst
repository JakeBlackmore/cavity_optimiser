Welcome to the documentation for the cavity optimiser
=====================================================
The cavity optimiser is a python module for computing the ideal cavity subject to constraints on cavity length and mirror volume.

Installation
------------
Install cavity optimiser by downloading the .whl file from GitHub and running:
    ``pip install cavity.whl``

on windows use:
    ``python -m pip install cavity.whl``

Usage
-----

Typical usage of this module requires the user-facing calculate module.
    ``from cavity import core``

An example calculations are shown in the python scripts on `GitHub`_.
